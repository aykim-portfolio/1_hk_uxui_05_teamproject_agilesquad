- 다음 멘토링 전에 결과물을 미리 공유할 수 있게 준비하기

코어 디자인
- 코어 디자인의 데이터는 타이틀을 쓸 때는 피그마 스타일에 등록이 되어 있어야 함
- 모바일로 갈 수 있는 변수를 등록을 해줘야 함
- 나머지 서브 페이지를 어떤 다른 두 분이 하신다고 하면 등록된 변수나 스타일을 가지고 작업을 하고 컴포넌트를 갖다 써야 함

콘텐츠 전략
- 콘텐츠 전략을 가지고 콘텐츠를 어떻게 잘 서비스를 제공할 건지에 포커스를 맞출 건지 UI를 어떻게 개선할 건지에 명확하게 와닿은 부분이 없음
- 어떤 부분을 하고 싶은지가 명확하게 안 나와 있는 것 같음

AI로 디자인 프로토 타입 제작
- 전체적인 흐름을 겪어보고 싶으시면 오히려 지금 AI로 디자인 프로토 타입을 뽑으셔서 러프하게 가는 게 맞으실 것 같음
- 전체적인 흐름을 알고 싶으시다면 그런 컴포넌트도 좀 러프하게 가져가셔도 됨

기사 뷰의 문제점
- 처음부터 끝까지의 달성해야 될 목표를 하나를 정하시는 게 더 좋을 것 같음
- 디자인을 해야 되는 범위도 기사 뷰 하나가 나옴
- 기사 뷰에 대해서 문제점을 파악하고 어떻게 개선을 할 건지 사용하는 사람을 더 특정 지을 건지를 더 디테일하게 정하셔야 될 것 같음

패널 기능
- 일반 사용자가 굳이 패널이 나온 화면을 봐야할까 의문이 듦
- 전문가 사용자들은 더 데이터를 보고 싶으실 거고 일반 사용자들은 뉴스 페이지를 일일이 다 하나하나 보는 게 아니라 속보로 나오는 것들을 다 요약해서 리서치 된 자료를 더 본다든지 이런 식으로 하실 것 같음

한경 뉴스의 강점
- 한국 경제에서 내보내고 있는 콘텐츠는 정제된 데이터임. 정제된 데이터를 갖고 있는 것에 대해서 강점이라고 생각함
- 디지털화가 됐기 때문에 신문 구독률은 떨어지지만 인터넷에서 볼 수 있는 뉴스들은 속보성이 강하고 흥미거리가 많고 휘발성이 너무 강함
- 신문에서 나온 것들은 편집 사항들을 다 거쳐서 정지된 데이터가 나오기 때문에 대기업 CEO들이랑 상위 1%는 신문을 꼭 본다는 얘기가 있음
- 유료 서비스에 담긴 뉴스 기사 같은 것들도 그런 부분들이 강점임

환경 프리미엄 라인
- 환경 프리미엄 라인에서 서비스를 제공할 부분을 일부 해서 API를 만들어 달라고 함
- 환경 프리미엄 라인 안에 UI를 만들고 에픽 AI 기능 만큼 다 서비스되고 있지는 않음
- 환경 프리미엄 라인에서 써보다가 더 쓰고 싶으면 에픽 AI 결제하라고 함

광고 페이지와 기사 페이지
- 광고에 대한 고민도 있고 자체 서비스를 유료화 유 구독으로 끌어오는 게 수익이 날 수 있을 것 같음
- 기존 프리미엄 회원 같은 경우에는 광고 페이지들보다 원래 있던 기능들을 더 끌어와서 사용해서 B2C로 전환을 할 수 있게 기사 정신에서 노출을 하다가 의도를 했었던 건데 너무 많이 섞이다 보니까 잘 보이지 않는 것 같음
- AI를 쓰는 데에 있어서 비용 문제에 고민이 많음
- 논리 근거를 가지고 접근해야 사용자한테 더 먹히겠다라고 하는 건 거지 답은 사용자가 써줄 거냐 그거에 대한 피드백을 빨리 받아야 되는 것임

기획자의 요건
- 피그마에서 퍼블리셔라든지 프론트엔드 개발자가 설계를 하는 뇌를 디자이너도 갖고 있어야 함
- 기획자가 협업하는 사람들한테 내 생각을 어떻게 표현할 건가 해서 와이어 프레임을 예전에는 더 이상 대체할 수 있는 게 없으니까 PPT나 피그 잼이나 그런 거를 써서 이렇게 했는데 지금은 GPT한테 UI 만들어줘 하면 다 만들어버리니까 그거 가지고 얘기하면 소통이 더 잘 됨
- 기획의 요건을 더 정교하게 적는 게 중요함
